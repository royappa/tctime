About TcTime TcTime provides these features related to Topcoder.com algorithm programming contests:
  • Given the points awarded for solving a problem, show how much time the coder took to solve it.
  • Given the time taken to submit a problem, show how many points it will score.
The calculated information assumes that the problem's first submission is correct.
Resubmitted problems incur penalties, which are not calculated here. For full details, see the Topcoder scoring formula. Developed using:
  • Ionic 2: ionicframework.com
  • AngularJS: angular.io
  • Firebase: firebase.google.com
