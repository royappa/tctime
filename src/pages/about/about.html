About Topcoder Time TcTime provides these features related to Topcoder algorithm programming contests:
  • Given the points scored on a problem, find out how much time it took.
  • Given the time scored on a problem, find out how many points it will score.

The app only counts the first submission. If you resubmit a problem, a penalty is applied and that is not simulated here.
